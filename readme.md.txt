# 🏋️‍♂️ App de Avaliação Física

Aplicativo profissional para personal trainers realizarem avaliações físicas completas, com visual dark, offline e geração de PDF.

## 🚀 Funcionalidades

- Entrevista inicial com dados do aluno
- Cálculo de TMB e NDC com base em idade, peso, altura, sexo e nível de atividade
- Métodos de dobras cutâneas (3, 4 ou 7)
- Circunferências opcionais
- Geração de relatório PDF dark com assinatura do profissional
- Armazenamento local (offline-first)

## 🛠 Tecnologias

- React Native (Expo)
- Styled Components
- Context API
- TypeScript
- React Native SVG + ViewShot (para gráficos)
- Expo Print & Sharing (para PDF)

## 📲 Como rodar

```bash
npm install
npx expo start
